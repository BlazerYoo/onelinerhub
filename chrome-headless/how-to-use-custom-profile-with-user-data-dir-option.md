# How to use custom profile with user-data-dir option

```js
google-chrome --headless --user-data-dir=/path/to/user2
```

- `--headless` - will launch chrome in [headless mode](https://developers.google.com/web/updates/2017/04/headless-chrome#cli)
- `--user-data-dir` - path to Chrome profile data dir to use
- `/path/to/user2` - sample path to use for Chrome profile data
