# Convert int to string

```lua
str = tostring(num)
```

- `str` - string variable that will contain result
- `tostring` - converts given value to string
- `num` - integer to convert to string

group: convert_str

## Example: 
```lua
str = tostring(123)
print(type(str))
print(str)
```
```
string
123

```
