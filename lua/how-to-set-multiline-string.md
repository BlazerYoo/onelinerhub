# How to set multiline string

```lua
str = [[
Hi,
I
am
multiline!]]
```

- `str =` - multi-line string
- `[[` - beginning of multi-line string
- `]]` - end of multi-line string

group: strings

## Example: 
```lua
str = [[
Hi,
I
am
multiline!]]

print(str)
```
