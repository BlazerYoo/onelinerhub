# Simple script example

```lua
some_var = 'all!'
print('Hi ' .. some_var)
```

- `some_var` - declare some variable
- `'all!'` - value of the variable declared
- `print` - outputs given text 
- `..` - join (concatenate) string on the left (`Hi `) with a string on the right (`some_var` variable)
