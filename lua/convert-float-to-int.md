# Convert float to int

```lua
y = math.floor(x)
```

- `y` - will contain resulting integer
- `x` - float value
- `math.floor` - will return largest integer that is small than given float number

group: convert_num

## Example: 
```lua
x = 16.2
y = math.floor(x)
print(y)
```
```
16

```
