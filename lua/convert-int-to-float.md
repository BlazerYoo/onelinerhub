# Convert int to float

```lua
y = x + .0
```

- `y` - will contain resulting float
- `x` - integer value
- `+ .0` - adds zero fraction to convert given number to float

group: convert_num

## Example: 
```lua
x = 25
y = x + .0
print(y)
```
```
25.0

```
