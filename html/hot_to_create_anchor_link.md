# Hot to create anchor link

```html
<a href="#anc">link to anchor</a>
<h2 id="anc">Title</h2>
```

- `href="#anc"` - link to an anchored element with `anc` id
- `h2 id="anc">` - anchored element, click of a previous link will scroll to the top of this element
