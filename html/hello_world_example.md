# Hello world example

```html
<!DOCTYPE html>
<html>
  <head><title>Hi</title></head>
  <body>
    Hello world!
  </body>
</html>
```

- `DOCTYPE` - define type of this document (html in our case)
- `title` - specify title of the browser window (or tab)
- `body` - main html content goes here
