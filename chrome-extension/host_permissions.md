# Specify host permissions

```json
{
  "name": "My",
  "manifest_version": 3,
  // ...
  "host_permissions": [
    "*://*/"
  ],
  // ...
}
```

- `"host_permissions"` - list hosts (domains) that extension will have access to
- `\*://\*/` - our extensions will be allowed to access all websites
