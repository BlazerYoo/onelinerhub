# Allow notifications permissions

```json
{
  "name": "My",
  // ...
  "permissions": [
    "notifications"
   ],
  // ...
}
```

- `permissions` - specify extension permissions list
- `notifications` - enable [notifications API](https://developer.chrome.com/docs/extensions/reference/notifications/) permissions
