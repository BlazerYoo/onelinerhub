# Allow cookies permissions

```json
{
  "name": "My",
  // ...
  "permissions": [
    "cookies",
    "*://*"
   ],
  // ...
}
```

- `permissions` - specify extension permissions list
- `cookies` - allow access to cookies
- `"\*://\*"` - access cookies on any website
