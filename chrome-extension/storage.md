# Allow storage permissions

```json
{
  "name": "My",
  // ...
  "permissions": [
    "storage"
  ],
  // ...
}
```

- `"permissions"` - specify what should be allowed for extension
- `"storage"` - allow access to [chrome.storage](https://developer.chrome.com/docs/extensions/reference/storage/)
