# Search files recursively with certain extensions only

```bash
grep 'text' -r --include *.txt /path/to/search/
```

- `'text'` - text to search
- `-r` - use recursive search
- `--include *.txt` - search only files with ```txt``` extension
- `/path/to/search/` - directory to search in
