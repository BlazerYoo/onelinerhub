# How to get angle from complex number

```python
import numpy as np
za = np.angle(3+4j)
```

- `3` - real part
- `4` - imaginary part
- `.angle` - computes angle of the complex argument
- `za` - will contain computed value in radians

group: complex

## Example: 
```python
import numpy as np
z = 3+4j
za = np.angle(z)
print(za)
```
```
0.9272952180016122

```
