# Convert degrees to radians

```python
import numpy as np
rad = np.radians(90)
```

- `import numpy as np` - load [lib:Numpy module](/python-numpy/how-to-install-python-numpy-lib) for Python
- `.radians(` - returns radians from given degrees value

group: rad_deg

## Example: 
```python
import numpy as np
rad = np.radians(90)
```
