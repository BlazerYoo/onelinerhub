# Convert radians to degrees

```python
import numpy as np
deg = np.degrees(1.57)
```

- `import numpy as np` - load [lib:Numpy module](/python-numpy/how-to-install-python-numpy-lib) for Python
- `.degrees(` - returns degrees from given radians value

group: rad_deg

## Example: 
```python
import numpy as np
deg = np.degrees(1.57)
print(deg)
```
