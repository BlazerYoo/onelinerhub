# How to get imaginary part from complex number

```python
z = 3 + 4j
real = z.imag
```

- `z` - declared complex number
- `3` - real part
- `4` - imaginary part
- `z.imag` - will return imaginary part of complex number

group: complex

## Example: 
```python
z = 3+4j
print(z.imag)
```
```
4.0

```
