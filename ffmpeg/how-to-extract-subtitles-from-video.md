# How to extract subtitles from video

```bash
ffmpeg -txt_format text -i in.mkv out.srt
```

- `-i in.mkv` - input video file
- `-txt_format text` - set output format to text (for subtitles extraction)
- `out.srt` - resulting subtitles file
