# How to delete local tag

```bash
git tag -d v0.7
```

- `git` - base command
- `tag` - manage tags
- `v0.7` - local tag name to delete
- `-d` - delete specified tag

group: tags

## Example: 
```bash
git tag -d v0.7
```
```
Deleted tag 'v0.7' (was a5b9df3)
```
