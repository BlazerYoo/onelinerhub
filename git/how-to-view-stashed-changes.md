# How to view stashed changes

```bash
git stash show
```

- `git` - base command
- `stash` - manage stash storage
- `show` - shows changes in stash

group: stash

## Example: 
```bash
git stash show
```
```
 test | 1 -
 1 file changed, 1 deletion(-)
```
