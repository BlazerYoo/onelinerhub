# How to get remote branch

```bash
git fetch origin branch4
```

- `git` - base command
- `fetch` - update data from remote
- `origin` - remote name
- `branch4` - branch name

group: branch_remote

## Example: 
```bash
git fetch origin test3
```
```
 * branch            test3   -> FETCH_HEAD
```
