# How to delete remote tag

```bash
git push origin --delete v0.7
```

- `git` - base command
- `push` - push local changes to remote
- `origin` - name of the remote to push changes to
- `--delete` - delete specified tag
- `v0.7` - tag name to delete

group: tags

## Example: 
```bash
git push origin --delete v0.7
```
