# How to delete local branch

```bash
git branch -d test
```

- `git` - base command
- `branch` - manipulate branches
- `-d` - remote specified local branch
- `test` - name of the local branch to delete

group: delete_branch

## Example: 
```bash
git branch -d test
```
```
Deleted branch test (was d7ded3f).
```
