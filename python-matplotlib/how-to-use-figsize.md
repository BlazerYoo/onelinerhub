# How to use figsize

### Using `figsize()` allows specifying figure size in inches:

```python
import matplotlib.pyplot as plt
plt.figure(figsize=(2, 2), dpi=80)
plt.plot([4,2,11,2,1,20,25])
plt.show()
```
